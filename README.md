# AgriNusa - Agriculture Website Template
![Preview](https://cdn.phototourl.com/free/2026-06-25-971f51ba-0ba6-4500-ac48-db00c145112a.jpg)

A clean, modern, and fully responsive agriculture website template built with HTML, CSS, and JavaScript. Perfect for farming businesses, agricultural stores, or farmers who want a professional online presence.

![Preview 2](https://cdn.phototourl.com/free/2026-06-25-4279dda2-4f79-4a33-ad14-f7630aef910d.jpg)

## Features
- Fully responsive (mobile-first)
- 5 complete pages (Home, About, Products, Services, Contact)
- Product filter by category
- Contact form with feedback
- Smooth scroll animations
- Loading screen
- Back to Top button
- SEO friendly
- Clean folder structure

## Technologies
- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (ES6)
- Font Awesome Icons
- Google Fonts
